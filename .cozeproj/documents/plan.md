# Ozon ERP 选品看板 UI 重新设计方案

## 概述
将选品看板重新设计为**数据密集型深色主题**，采用紧凑表格布局、高信息密度展示。优化市场信号与已选品列表的视觉层级和交互体验。

## 技术方案
| 维度 | 选择 | 理由 |
|------|------|------|
| 设计风格 | 数据密集型深色主题 | 适配 B2B 运营场景，提高信息获取效率 |
| 主色调 | 品牌蓝 #2F6BFF + 深色背景 | 保持品牌识别度 |
| 布局 | 紧凑表格 + 卡片混合 | 兼顾数据浏览与详情查看 |
| 组件库 | shadcn/ui + 自定义深色主题 | 统一组件风格 |

## 功能模块

### 1. 选品看板主页
- 顶部：Tab 切换（市场信号 / 已选品）+ 筛选器 + 刷新按钮
- 主区域：数据表格 / 卡片列表
- 右侧：快捷操作面板（批量操作、导出）

### 2. 市场信号 Tab
- 紧凑数据卡片网格（来源Badge、价格、卖家数、评分）
- 快速加入选品按钮
- 图片缩略图（可点击查看大图）

### 3. 已选品 Tab
- 表格列表视图
- 支持排序、筛选、批量移除
- 状态标签（待审核/已采购/已上架）

## 页面规格

### @page(/selection) 选品看板

**核心职责**：展示和管理市场信号数据，支持快速选品决策

**布局**：
- 顶部导航栏：Logo + 页面标题
- Tab 区域：市场信号 / 已选品 切换
- 工具栏：筛选下拉 + 刷新按钮
- 主内容区：数据卡片网格或表格

**状态**：
- 空态：暂无数据提示 + 引导采集
- 加载态：骨架屏
- 错误态：重试按钮
