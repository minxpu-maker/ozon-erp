  - 模式B「系统推荐」：系统主动扫描全平台，推荐当前最值得做的类目/商品

## 二、选品算法决策

- **决策人**：飞羽综合团队方案，林中飞确认

### D-011：评分方法 — AHP-TOPSIS替代线性加权
- **决策**：AHP算权重 + TOPSIS算贴近度排序
- **说明**：线性加权三大硬伤——权重拍脑袋、维度互补偿、无法处理天然冲突
- **决策人**：飞羽提议，团队认可，林中飞确认

### D-012：语义评分通道 — Embedding+LightGBM（一期降级）
- **决策**：产品标题/描述 → Embedding → 特征拼接 → LightGBM
- **一期降级**：无标注数据无法训练LightGBM，降级为Embedding余弦相似度替代；积累3个月后切换LightGBM
- **说明**：LLM不直接打分（数字敏感度差），Embedding做特征提取+LightGBM做数值推理
- **决策人**：claude code提议，团队认可

### D-013：时序预测 — Prophet+俄罗斯假期日历

- **一期冷启动**：无历史销量数据时，用Ozon平台级类目趋势替代SKU级预测；积累2周数据后启用SKU级预测
- **说明**：俄罗斯季节性极强（冬季6个月+新年峰值占全年GMV极高），yearly_seasonality=15
- **决策人**：深度调研推荐，团队认可

### D-014：EAC认证 — 按卖家类型条件触发
